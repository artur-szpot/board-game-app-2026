# 13 Implement games list/detail screens

## Description
Implement the frontend screens for listing games and viewing game details.

## Acceptance criteria
- `LIST_GAMES` and `VIEW_GAME` screens are implemented.
- Screens display game details and related tags, locations, scoring schemas, and helpers.
- Screens support opening edit flows via frame actions.
- Screen data is loaded from `/game-api/games` endpoints.

This ticket depends on locations, tags, scoring schemas, and helpers backend support.
