# 11 Implement generic screen templates

## Description
Add the reusable frontend templates for FORM, SEARCH, and OPTIONS screens.

## Acceptance criteria
- Generic FORM, SEARCH, and OPTIONS components exist.
- Minimal UI controls are implemented for MVP.
- Screens can accept `dataType`, selection limits, and action callbacks.
- Templates can render forms, search results, and option buttons.

## Notes
These templates are required before actual entity screens are built.
