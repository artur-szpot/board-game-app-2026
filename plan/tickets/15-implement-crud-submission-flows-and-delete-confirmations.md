# 15 Implement CRUD submission flows and delete confirmations

## Description
Implement frontend submission and deletion flows for MVP CRUD operations.

## Acceptance criteria
- Forms submit data to the backend via the appropriate `/game-api/*` endpoint.
- Delete actions show a confirmation prompt before execution.
- Minimal loading state is shown during API requests.
- Empty-state messaging appears when no results are found.

## Notes
This ticket depends on backend CRUD endpoints and frontend screens.
