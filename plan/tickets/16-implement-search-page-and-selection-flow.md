# 16 Implement search page and selection flow

## Description
Implement the frontend search page and result selection workflow.

## Acceptance criteria
- Frontend uses `/game-api/search` for lookups.
- The `type` parameter is required and passed as an array with at least one type.
- Search results display a combined list of typed items.
- `No results found` displays when no matches are returned.

## Notes
This ticket depends on the shared search backend endpoint.
